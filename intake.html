04 Files & Statements

Required
Distributor Royalty Statement (CSV or PDF)
Export from your distributor: Earnings → All Time → Download. Include every available month. DistroKid users: "SEE EXCRUCIATING DETAIL" → choose a month → download TSV.
Required
Screenshot — Distributor Dashboard Overview
The main earnings/royalties page showing: Balance, Pending Earnings, Paid Earnings, and the full earnings history table. This documents the payment state.
Recommended
Screenshot — Each Release Detail Page
Click into each release in your distributor and screenshot the page. We need to see any warning icons, copyright flags, or ACRCloud notices.
Recommended
PRO Royalty Statements (ASCAP, BMI, etc.)
My Account → Statements → Download. Last 8 quarters as PDF or Excel. ASCAP: electronic download available. BMI: Excel export available.
Recommended
MLC Royalty Statements
Log into themlc.com → Member Hub → Royalty Statements → Download CSV for each available month.
Recommended
Songtrust or Publishing Admin CSV
If using Songtrust, TuneCore Publishing, or similar: log in and export the all-data CSV from the Statements or Royalties section.
Recommended
Signed Split Sheets
For any co-written songs. Required to resolve ACRCloud copyright flags. PDF scan or photo is fine.
